# Zenithia — Item Icon Spec Sheet

## Style Guide

**Art Style:** Flat icon, RPG game item style, dark background (#1a1a2e), soft glow rim
**Size:** 128x128px (or 256x256 for high-res)
**Format:** PNG with transparent/dark background
**Consistency Prompt (copy this at the end of EVERY generate):**

```
Flat RPG game item icon, dark moody background, soft edge glow, detailed but clean, game UI style, no text, no watermark, centered composition
```
