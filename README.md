# Zenithia

A colorful MMORPG built on Solana blockchain.

## Features
- 3D world with boxy/Minecraft-style characters
- 5 unique classes: Guardian, Blade Dancer, Sage, Cleric, Shadow
- Dynamic monsters with unique behaviors
- Solana wallet integration for player identity
- NPC-driven story with "The Fracture" endgame twist

## Tech Stack
- **Client:** Three.js (3D), WebSocket (real-time)
- **Server:** Node.js, WebSocket
- **Blockchain:** Solana (Phantom wallet)
- **Database:** JSON persistence (MVP)

## Getting Started
```bash
npm install
node server.js
# Open http://localhost:2567
```
